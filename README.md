# Payloads
Web uygulama güvenliği testleri için hazırlanmış, çeşitli saldırı vektörlerini içeren pratik bir payload arşividr. Repo, penetration testing ve güvenlik araştırmalarında kullanılabilecek temel örnekleri içerir.

## İçerik
| Dosya | Açıklama | Kategori |
|-------|----------|----------|
| `sql_injection.txt` | Temel SQL enjeksiyon payload'ları | Veritabanı Güvenliği |
| `sql_login_bypass.txt` | Login bypass teknikleri | Kimlik Doğrulama |
| `sqlite_injection.txt` | SQLite özel enjeksiyonlar | Veritabanı Güvenliği |
| `xss.txt` | Cross-Site Scripting örnekleri | İstemci Tarafı |
| `svg-to-xss.svg` | SVG tabanlı XSS vektörü | Dosya Yükleme / XSS |
| `xxe.txt` | XML External Entity payload'ları | XML Güvenliği |
| `lfi.txt` | Local File Inclusion teknikleri | Dosya Dahil Etme |
| `command_injection.txt` | Komut enjeksiyonu örnekleri | Sunucu Tarafı |
| `xee.txt` | XML Entity Expansion örnekleri | DoS / XML Güvenliği |

## Kullanım Alanları
- Penetrasyon Testleri
- Güvenlik Eğitimi
- CTF Yarışmaları
- WAF Testleri

## Önemli Uyarı

&gt; Bu payload'lar yalnızca yasal ve etik sınırlar içinde kullanılmalıdır:
&gt; 
&gt; - Kendi uygulamalarınızda test yapın
&gt; - İzin aldığınız sistemlerde kullanın
&gt; - Eğitim ve araştırma amaçlı değerlendirin
&gt; - İzinsiz sistemlere saldırmayın
&gt; - Kötüye kullanımdan sorumluluk size aittir
